# Privacy Policy URL

The source version of the privacy policy is stored in the repository root:

```text
PRIVACY.md
```

GitHub URL:

```text
https://github.com/smeshchankin/izar4-padel-gpt/blob/main/PRIVACY.md
```

Alternative raw URL:

```text
https://raw.githubusercontent.com/smeshchankin/izar4-padel-gpt/main/PRIVACY.md
```

## GPT Builder Configuration

Use one of the public URLs above in GPT Builder under:

```text
Configure → Actions → Privacy policy URL
```

Recommended value:

```text
https://github.com/smeshchankin/izar4-padel-gpt/blob/main/PRIVACY.md
```

If GPT Builder does not accept the GitHub page URL, use:

```text
https://raw.githubusercontent.com/smeshchankin/izar4-padel-gpt/main/PRIVACY.md
```

## Notes

- The repository must be public.
- `PRIVACY.md` must exist in the `main` branch.
- If the policy changes, update `PRIVACY.md` in the repository.
- The repository copy is the source of truth for the privacy policy.
