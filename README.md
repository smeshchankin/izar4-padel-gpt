# IZAR4 Padel Booking GPT

Custom GPT configuration for booking the IZAR4 padel court via Actions API.

This repository contains:
- GPT name, description, and instructions
- OpenAPI schema for GPT Actions
- conversation starters
- booking and cancellation safety rules
- privacy policy and disclaimer documents
- manual test checklist
